
---
## A Recreation of MacOS
- Made in HTML5, CSS3 and Javascript
---
## Features
- Time (Seconds also can be viewed)
- Finder UI
- Day and Night wallpapers (after 6pm, it turns in to night wallpaper)
- Calender
- Notification banner
- Settings UI
- Terminal
- Apple Menu UI
- A Messenger app UI
- Right click (Only Refresh is available)
- Lock screen
- Log out/in user
- App Search
